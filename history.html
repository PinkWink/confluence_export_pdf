User Request

1. Confluence PDF 품질 개선 요청

Confluence Cloud의 기본 PDF exporter 출력이 이쁘지 않고, 특히 코드블록에 행번호가 보이지 않는 문제를 해결하고 싶다. MCP를 이용해 Confluence에 접속하여 페이지를 HTML/PDF로 변환할 수 있는지 문의.

  • Domain: https://pinkwink.atlassian.net/
  • Email: pinkwink.korea@gmail.com
  • API Token: 제공 완료
  • 출력 형식: HTML + PDF 둘 다
Claude Implementation

2. 접근 방법 설계 및 계획 수립 Plan

3가지 접근 방법(Confluence REST API + 스크립트, Confluence MCP Server, WebFetch) 중 REST API + Python 스크립트 방식을 선택.

  • conda 환경 confluence 생성 (Python 3.11)
  • 패키지: requests, beautifulsoup4, playwright, python-dotenv
  • Playwright Chromium으로 PDF 렌더링
Claude Implementation

3. 핵심 스크립트 작성 Done

confluence_export.py 작성 — 주요 기능:

  • Confluence REST API v2/v1 자동 전환
  • URL에서 page ID 자동 추출
  • 코드블록 행번호 추가 (<ac:structured-macro> → styled HTML)
  • 이미지 base64 인라인 임베딩
  • 패널, 이모티콘, 링크, 상태 뱃지 등 매크로 변환
  • Playwright Chromium으로 PDF 생성
Claude Implementation
